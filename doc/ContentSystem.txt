Веб-страницы образуются путём использования двух текстовых файлов: шаблона страницы и содержимого страницы.
Основной шаблон страницы находится в файле content/pageTemplate.html .
И шаблон и страницы могут содержать обращения к подстановочным переменным, например: $currentUserName$
Некоторые подстановочные переменные доступны на всех страницах, в то время как другие подстановочные переменные доступны только на некоторых страницах. За конкретные логические страницы отвечают классы в пакете hmania.AH, наследуемые от класса hmania.AH.ActionHandler. 
Подстановочные переменные доступные на всех страницах:
	$action$ : текущий action
	$currentUserName$ : нежёсткое отображаемое имя пользователя, если пользователь не идентифицирован, возвращает "log in"
	$currentUserAccess$ : нежёсткое отображаемое состояния доступа
	$userButtonLink$: URL для кнопки-ссылки с именем пользователя: в случае если пользователь идентифицирован указывает на страницу текущего пользователя, иначе указывает на страницу входа
Все файлы в папке content доступны в подстановочных переменных вида $file.name.ext$, где имя файла = name.ext. Например, содержимое файла content/style.css может быть подставлено в текст через переменную $file.style.css"
Предусмотрены специальные блоки в контент-тексте:
<title>Заголовок страницы</title> В блоке title можно указывать заголовок страницы.
<style>Стиль страницы</style> В блоке style можно указывать стили которые будут вставлены в блок html/head/style для данной страницы.
Перечисленные контент-блоки предназначены только для содержимого страницы. Шаблон страницы определяет как будут использованы контент-блоки с помощью подстановочных переменных $style$ и $title$, доступных только в тексте шаблона, но недоступных в тексте содержимого.
